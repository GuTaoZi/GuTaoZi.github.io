The notes of Evolutionary Reinforcement Learning：A Survey

Reinforcement Learning

  • Model-based
    • Estimate transition function and reward function
    • Establish a complete MDP
  • Model-free
    • Large sample training, no need to know two functions
    • policy-based
      • Trust Region Policy Optimization (TRPO)
      • Proximal Policy Optimization (PPO)
      • Asynchronous Advantage Actor-Critic (A3C)
      • Deep Deterministic Policy Gradient (DDPG)
      • Twin Delayed DDPG (TD3)
      • Soft Actor-Critic (SAC)
    • value-based
      • ​ Deep Q-Network (DQN)

Evolutionary Computation

Evolutionary Strategies

  • Canonical ES

    Variation -> evaluation -> selection

  • Covariance Matrix ES

    Adaptively update: mutation step size, covariance matrix

  • Natural ES

    Update distribution parameters by estimating search gradient

Integration with RL: policy search, exploration, reward shaping, hyperparameter optimization, meta-RL, multi-objective RL

Genetic Algorithms

selection -> crossover -> mutation

combinatorial optimization problems

Integration with RL: policy search, exploration, reward shaping, hyperparameter optimization, meta-RL, multi-objective RL

Genetic Programming

Search space: set of programs

Fitness: observing the behavior of program

Programs viewed as data: crossover & mutation

Data interpreted as programs: execution

Numerical optimization problems, problems with programmatic solutions

Integration with RL: policy search, reward shaping, meta-RL

EvoRL Frame

EC methods Encodings Variation Selection criteria Target Problem in RL
ES Real Mutation Performance Continuous optimization
GA Int, real, indirect Crossover, mutation Performance, diversity General optimization
GP Syntax free, executable graph Crossover, mutation Performance, interpretability Regression / control problems

asynchronous PBT

Hybrid methods
